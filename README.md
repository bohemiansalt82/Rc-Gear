# Rc-Gear (Fusion 360 Add-ins)

이 저장소는 Fusion 360에서 사용할 수 있는 정밀 타이밍 벨트 및 풀리 생성 도구를 포함하고 있습니다.

## 포함된 도구
1. **S3M Belt Generator**: 정밀 S3M 규격의 타이밍 벨트(Closed Loop)를 생성합니다.
2. **S3M Pulley Generator**: 벨트와 완벽하게 호환되는 타이밍 풀리를 생성합니다.

## 설치 방법
1. 이 저장소를 다운로드하거나 클론합니다.
2. Fusion 360에서 `Shift + S`를 눌러 **Scripts and Add-ins** 창을 엽니다.
3. **Add-ins** 탭에서 **+** 버튼을 눌러 각 폴더(`S3M_Belt_Generator_Addin`, `S3M_Pulley_Generator_Addin`)를 추가합니다.
4. 추가된 애드인을 선택하고 **Run**을 누릅니다.

## 사용법
- **SOLID** -> **CREATE** 패널 하단에 생성된 버튼을 클릭하여 매개변수(길이, 이빨 개수, 폭 등)를 입력합니다.
